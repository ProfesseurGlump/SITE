Les thèmes que vous trouverez sur cette page.

Mise en garde : je n'ai aucune prétention concernant le hacking. Mon but est uniquement pédagogique et ludique. Je ne cherche pas à pirater les ni à permettre à des gens de devenir des pirates. Je ne prétends pas être un grand technicien, je suis un grand débutant. Par contre je suis un débutant qui aime partager ses découvertes. A vous de juger si les documents, liens et autres informations que je délivre ici sont pertinentes, intéressantes, utiles ou pas.

Cette mise en garde étant faite, j'espère pouvoir accueillir des visiteurs bienveillants.

Voici un regroupement en 6 catégories qu'un hacker doit maîtriser s'il veut en être un véritable (emprunté au site hackademics.fr)

  1. Les types de langage ( impératif, POO, embarqué, logique)
  2. La modélisation ( singletons, UML, Merise)
  3. La compilation ( compilateur, assemblage, linkage, appels)
  4. Les langages orientés Web (finalité, architecture, HTML/PHP/JS)
  5. Les langages orientés logiciel (finalité, architecture, C/ C++/ Java(mixte)/ VB)
  6. Les langages orientés modulaire (finalité, architecture, Python)

Il peut être également intéressant de consulter ce site afin d'avoir un autre avis sur le sujet.

Pour ma part je ferais la liste suivante des langages informatiques à maîtriser si l'on veut devenir un hacker :

  • Le langage C et pour l'étudier je recommande ce site ou encore celui-ci
  • Le langage HTML car c'est celui pour écrire les pages web (du coup on peut inclure également le PHP et le JS (JavaScript))
  • Le langage Java
  • Le langage Lisp car c'est un langage fonctionnel qui a beaucoup servi en intelligence artificielle (et qui sert encore malgré son âge vénérable) et en particulier pour l'utilisation de l'éditeur Emacs qui est totalement paramétrable en Emacs Lisp.
  • Le langage Python peut être un bon langage pour débuter, je recommanderais ce site pour apprendre à programmer.
  • Le langage Perl (utile pour écrire des scripts).
  • Le langage R pour faire des statistiques.
  • Le langage TeX pour composer des documents (en particulier avec des formules mathématiques).
