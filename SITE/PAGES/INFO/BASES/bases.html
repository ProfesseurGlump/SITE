Les thèmes que vous trouverez sur cette page.

Vous trouverez ici 7 grandes catégories (très subjectives) concernant ce que j'appelle les bases en informatique.

Vous êtes invité à me faire part des sujets qui vous préoccuppent afin d'enrichir cette rubrique.

Voici donc les 7 catégories tant attendues :

  1. L'internet et plus spécialement le web et la messagerie électronique.

    On présentera quelques exemples de :

    • navigateurs web
    • clients de messageries électroniques
    • d'hébergeurs de webmail

  2. Les recherches sur le web

    On présentera :

    • une alternative à Google
    • comment demander de l'aide sur les forums

  3. Navigation dans votre système

    On présentera :

    • ce qu'est un système d'exploitation ainsi que des exemples
    • l'approche graphique
    • l'approche en ligne de commande

  4. Bonnes pratiques sécuritaires

    On présentera :

    • comment créer un mot de passe sûr et simple à retenir
    • pourquoi il faut être vigilant avec les informations que l'on diffuse sur le web
    • une première définition d'un Hacker

  5. Vocabulaire avancé (un peu de théorie)

    On présentera :

    • les langages
    • les informations
    • les machines
    • les algorithmes

  6. Eléments de HARDWARE (un peu de matériel)

    On présentera :

    • la carte mère
    • le processeurs
    • le(s) disque(s) dur(s)
    • la mémoire RAM
    • les périphériques externes

  7. Comment créer un site web

    On présentera :

    • le langage HTML (pour structurer une page web)
    • le langage CSS (pour décorer une page web)
    • les langages SQL (pour gérer une base de données)
    • le langage JS (pour rendre dynamique une page web)
    • le langage PHP
