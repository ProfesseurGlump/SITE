Introduction

Bien que le développement web a changé au fil des ans, les réels détails de base de la création d'une page Web sont restés les mêmes. Vous créez un document et vous le mettez sur le Web pour que les gens le voient. Bien sûr, pour mettre quelque choses sur le Web vous avez besoin d'apprendre les langages spéciaux qui sont parlés sur le web. Nous parlons des langages qui sont utilisés pour créer des documents et sites web.
